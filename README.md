## Description

Ce projet est une application fullstack permettant de gérer des utilisateurs (CRUD complet).

- **Backend** : NestJS avec TypeORM et SQLite
- **Frontend** : React avec TypeScript (Vite)

## Fonctionnalités

- Afficher tous les utilisateurs (getAllUsers)
- Voir les détails d'un utilisateur (getOneUser)
- Créer un nouvel utilisateur (createUser)
- Modifier un utilisateur existant (updateUser)
- Supprimer un utilisateur (deleteUser)

## Installation et lancement

### Backend
```bash
npm install
npm run start:dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

L'application sera accessible sur http://localhost:5173
